# Fotos reales pendientes de pedir al cliente

Ya se reemplazó por fotos reales todo lo de **van**, **limpieza de sensor**,
**cobertura** y **equipo trabajando**. Quedan 3 zonas que aún usan fotos de
stock porque no había foto real en la carpeta. Cuando el cliente las envíe,
se reemplazan igual.

## 1. Tienda — fotos de producto (8)
Una foto por producto, fondo limpio (idealmente blanco o neutro), ~1200×800.
El cliente las puede subir él mismo desde el panel `/admin` cuando tenga el
stock real. Productos demo actuales:
- Canon EOS R6 — cuerpo
- Nikon D850 — cuerpo
- Fujifilm X-T4 — cuerpo
- Sony FE 50mm f/1.8 — lente
- Canon EF 24-105mm f/4L — lente
- Profoto B10 — flash
- Trípode de fibra de carbono
- Kit de tarjetas SD

## 2. Producciones de foto/video — 5 categorías
Una foto representativa de cada servicio (las que ellos mismos hayan
producido es ideal — muestra su trabajo real):
- Gastronomía / restaurante
- Producto / e-commerce
- Retratos / headshots
- Eventos / cobertura
- Naturaleza / paisaje

(Hay 102 videos en la carpeta `Pic/VideosMPC` — si quieren, de ahí se pueden
sacar fotogramas para algunas de estas categorías.)

## 3. Equipo / About
- 1 headshot del CEO **Orlando Untoria** (vertical, ~900×1100)
- Opcional: foto grupal del equipo

---

### Formato recomendado para todas
JPG, lado largo ≥1600px, buena luz. Se optimizan al integrarlas.
Las de tienda las puede cargar el cliente solo desde `/admin`; las de
producciones y equipo las integro yo en el código.
